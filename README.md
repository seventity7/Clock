### 🔧 Dalamud Installation

| Step | Action |
| --- | --- |
| 1️⃣ | Open the Dalamud plugin list with `/xlplugins` |
| 2️⃣ | Search for `Clock` |
| 3️⃣ | Click **Install** |
| 4️⃣ | Enable the plugin |
| ✔ | Open settings with `/clock` and control the overlay with `/clock on` / `/clock off` |

</div>

> [!TIP]
> Clock is intended to be installed through the in-game Dalamud plugin installer only.

<a id="how-it-works"></a>

### 🧠 How It Works

Clock shows an in-game overlay with the selected timezone and format. You can keep it simple, use a digital/tech style, or customize the entire look in the `Appearance` tab.

### 🔹 Timezone Display

**Examples:**
| Option | Result |
| --- | --- |
| `America/New_York` | Shows New York time |
| `Japan Standard Time` | Shows Japan time |
| `UTC` | Shows universal time |

You can also keep favorite timezones for faster switching.

> [!NOTE]
> Use this when you want to track another timezone while playing.

### 🔹 Profiles & Presets

Clock supports saved profiles and built-in presets/themes so you can quickly switch between different looks.

| Preset Examples | Description |
| --- | --- |
| `Digital` | Digital clock style |
| `Tech` | Technology-style display |
| `Minimal` | Simple clean overlay |
| `Raid Minimal` | Compact setup for gameplay |
| `Casino Gold` | Gold-themed visual style |

There is currently `13+` themes to choose from.

---

### 🔹 Chat Timestamp & Time Conversion

Clock can add a custom timestamp to chat messages and also includes an optional experimental chat time-conversion feature.

Chat Time Conversion automatically detects time values written in chat and helps convert them into your configured timezone/time format.
This makes it easier to understand scheduled events, party plans, raid times, FFXIV event times or any shared time without manually calculating timezone differences.
It is useful when playing with people from different regions, helping everyone read the same time more clearly and avoid confusion.

> [!WARNING]
> Chat time-conversion is experimental and may conflict with other chat-related plugins.

<a id="commands"></a>

### ⌨️ Commands

<div align="center">

| Command | Description |
| --- | --- |
| `/clock` | Open the settings window |
| `/clock on` | Show the clock overlay |
| `/clock off` | Hide the clock overlay |
| `/clocksettings` | Open Clock settings/customizations |
| `/alarms` | Open alarm overlay |
| `/clock timezone <timezone>` | Change the main clock timezone |
| `/clock format 12\|24\|12s\|24s\|weekday\|date` | Change the time format |
| `/clock colon default\|always\|hidden\|slow\|fast` | Change colon animation |
| `/clock layout horizontal\|vertical` | Change the active profile layout |
| `/clock <timezone> to <timezone>` | Compare the current time between two timezones |
| `/clock lock` | Lock clock movement |
| `/clock unlock` | Unlock clock movement |
| `/clock profile next\|list\|set <n>\|add <name>\|rename <name>\|delete` | Manage clock profiles |

</div>
